# Stefan's dotfiles

## How to get started

```
bash ~/path/to/dotfiles/install
```

## What's included

- Alfred preferences
- brew dependencies
- iTerm2 config
- npm config and dependencies 
- shell functions
- VSCode config and plugins
- zsh config and aliases

## Notes

### Alfred workflows

Alfred workflows are included in `.gitignore`. These will be installed in `node.sh` via npm packages.
